# Bookshop API

This is my Bookshop API project.  
It is made using Node.js, Express, and Axios.  

## Files

- `server.js` - main file to run the backend server  
- `routes.js` - all the routes for the tasks 1 to 9  
- `books.js` - data for 5 popular books  
- `users.js` - stores registered users  
- `client.js` - axios requests for tasks 10 to 13  
- `package.json` - dependencies and start script  

## How to run

1. Make sure Node.js is installed.  
2. Open terminal and go to project folder.  
3. Run `npm install` to install dependencies.  
4. Run `npm start` to start the server.  
5. Open another terminal and run `node client.js` to test axios requests.  

## Book List

This project has 5 popular books:

- Twilight by Stephenie Meyer  
- A Game of Thrones by George R. R. Martin  
- Harry Potter and the Sorcerer's Stone by J.K. Rowling  
- To Kill a Mockingbird by Harper Lee  
- Pride and Prejudice by Jane Austen  

## Tasks

- Task 1 to 9: backend API (get books, get by ISBN, author, title, reviews, register, login, add/delete review)  
- Task 10 to 13: axios requests to test the API
